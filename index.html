About Me

Hello and welcome to my portfolio! My name is Sharjil Mohsin and I'm currently in my 4th year of the Software Engineering and Management program at McMaster University in Hamilton, Ontario, Canada.

My interest in the world of technology started when I took computer science in high school. It was there when I realized coding was something I was good at and ended up being interested in it. This was the turning point in my life that helped me choose software engineering as my career choice. I have a lot of interest in the emerging world of new technologies such as the rapid expansion of artificial intelligence, quantum computing, the introduction of the 5G network and Starlink. By tackling real-world problems through my side projects, I desire to improve my understanding of the changing technologies to contribute to a brighter and cleaner world.

Aside from my passion for programming, some of my other interests includes biking, swimming, playing video games, photography, and playing soccer, basketball, badminton, and ping-pong.

Image Repository

Image Repository

Technology Used: Python, Django, PostgreSQL, Google Cloud Vision API, Heroku, AWS

This image repository application is built using Django and Python. Users can upload images, download, archive, and search for images that have been uploaded into the repo. Using object detection (cvlib by default and Google Vision API if the credentials are provided), the image will also have suggested tags attached to it that users can use to find the uploaded images. This application uses Heroku and AWS S3 for image hosting and scalability.
